# AI Recruitment System Frontend

Professional SaaS-style Next.js App Router foundation for the AI Recruitment System.

## Stack
- Next.js App Router
- Tailwind CSS
- shadcn/ui ready scaffold
- Axios
- TanStack Query
- Lucide icons

## Environment
Create `frontend/.env.local` from `.env.example`:

```env
NEXT_PUBLIC_API_BASE_URL=http://localhost:8000/api
```

## Scripts
```bash
npm install
npm run dev
```

## Routes
- `/` dashboard placeholder
- `/candidates`
- `/jobs`
- `/rankings`
- `/ai-tools`
- `/automation`

## Notes
This step intentionally ships only the UI foundation and service scaffolding. Real API integration is reserved for Step 9.
